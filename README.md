# Anatomic Laterality Quiz

Browser-based laterality reaction-time quiz for anatomy images.

## How to Run

Open `laterality_quiz.html` in a web browser.

The quiz shows 10 randomized images, waits for a Left or Right choice, times the response, reveals the correct answer after each choice, and shows the average reaction speed at the end.
